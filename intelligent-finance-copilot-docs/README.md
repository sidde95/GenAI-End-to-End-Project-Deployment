# Intelligent Finance Copilot Documentation

This folder contains the complete documentation set for the **Intelligent Finance Copilot** application.

The application combines key capabilities from modern open-source finance products into a focused, minimal-functionality implementation:
- Income and expense tracking
- Budget planning and monitoring
- Portfolio tracking
- Net-worth and financial health dashboard
- Reporting and insights
