# User Stories

## Authentication
- As a user, I want to register with email and password so I can securely access my data.
- As a user, I want to log in and out reliably so my account remains protected.

## Transactions
- As a user, I want to add income and expense transactions so I can maintain my financial ledger.
- As a user, I want to edit or delete transactions so I can correct mistakes.
- As a user, I want to categorize transactions so I can analyze spending patterns.

## Budgets
- As a user, I want to set budget limits per category so I can control spending.
- As a user, I want to view budget utilization so I can detect overspending early.

## Portfolio
- As a user, I want to add holdings so I can track investment value.
- As a user, I want to see gain/loss metrics so I can evaluate performance.

## Dashboard & Reports
- As a user, I want a financial summary dashboard so I can assess overall health quickly.
- As a user, I want period-based reports so I can review financial behavior.

## Reliability & Security
- As a user, I want my data to be stored securely so I trust the platform.
- As a user, I want consistent application behavior so I can depend on it.
