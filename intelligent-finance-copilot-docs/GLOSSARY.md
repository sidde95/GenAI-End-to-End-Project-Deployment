# Glossary

- **Asset Allocation:** Distribution of portfolio value across asset classes.
- **Budget Utilization:** Ratio of category spending to budget limit.
- **Cash Flow:** Net movement of money in and out over a period.
- **Cost Basis:** Original value used to compute gains/losses.
- **Holding:** A recorded investment position.
- **Net Worth:** Assets minus liabilities.
- **Portfolio:** Collection of investment holdings.
- **Unrealized Gain/Loss:** Value change of an asset not yet sold.
- **Period Key:** Normalized month identifier (YYYY-MM).
- **Read Model:** Pre-aggregated/optimized data representation for UI queries.
