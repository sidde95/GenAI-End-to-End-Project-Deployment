# FAQ

## What does this application do?
It provides personal finance tracking through transactions, budgets, portfolio monitoring, and dashboard insights.

## Is this application a financial advisor?
No. It is a financial management and analysis tool. It does not provide regulated investment advice.

## Does it support automatic bank sync?
Not in the core scope. Data entry is manual-first.

## Can users track investments?
Yes. Users can maintain holdings and view valuation summaries.

## Is deployment-ready documentation included?
Yes. This folder includes architecture, deployment, CI/CD, operations, security, and testing documentation.

## Can this be extended with AI features?
Yes. The architecture and backlog explicitly support future intelligent capabilities.
