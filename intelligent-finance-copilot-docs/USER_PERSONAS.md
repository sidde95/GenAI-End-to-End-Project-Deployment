# User Personas

## Persona 1: Budget-Conscious Professional
- Wants clear monthly spending limits
- Needs fast transaction entry
- Cares about category-level visibility
- Needs over-budget alerts

## Persona 2: DIY Investor
- Tracks equities/crypto/funds manually
- Wants portfolio value and allocation view
- Needs gain/loss summaries

## Persona 3: Financial Organizer
- Wants one place for all personal finance records
- Reviews historical trends and report summaries
- Prefers easy exports and clear terminology

## Persona 4: Reviewer/Manager
- Evaluates implementation quality and operational readiness
- Needs clear documentation and predictable behavior

## Shared Expectations
- Data privacy
- Accurate calculations
- Responsive UI
- Reliable deployment and uptime
